# Inventario Android - Ferreteria Zamora

Aplicacion Android para gestion de inventario con sincronizacion al backend.

## Tecnologias
- Kotlin
- SQLite (almacenamiento local)
- Material Design 3
- Retrofit 2 (sincronizacion con backend)
- RecyclerView

## Funcionalidades
- CRUD completo de productos (offline con SQLite)
- Listado con RecyclerView y MaterialCardView
- Sincronizacion con backend Spring Boot via REST API
- Interfaz Material Design 3

## Requisitos
- Android Studio
- Min SDK: API 24 (Android 7.0)
- Target SDK: API 34
